# 📊 MS Excel SalesDashboard Project

An interactive Sales Dashboard built in Microsoft Excel to analyze and visualize sales performance using Pivot Tables, Pivot Charts, Slicers, and advanced Excel features. This project demonstrates how raw sales data can be transformed into meaningful business insights through interactive reporting.

---

## 📌 Project Overview

The objective of this project is to create a dynamic and user-friendly dashboard that enables users to monitor sales performance, identify trends, compare regions and products, and make data-driven decisions.

The dashboard provides an easy-to-understand overview of key business metrics by converting raw sales data into interactive visualizations.

---

## 🚀 Features

- Interactive Dashboard
- Dynamic Slicers for Data Filtering
- Pivot Tables & Pivot Charts
- Monthly Sales Trend Analysis
- Region-wise Sales Performance
- Product-wise Sales Analysis
- Salesperson Performance Comparison
- Clean & Professional Dashboard Layout
